[X] 試做彈性卡片

  • 替卡片設了 flex-grow: 1，有剩餘空間時，將這些空間分配給卡片以長大
  • 設定容器的最大寬度為 800px，當元素寬度大於容器寬度時，flex-wrap: wrap 會換行
  • 心得：
  • 想要做響應式卡片，寬度不足時，卡片掉下去，一行一張，但做不出來
  • 只做出了寬度不足時會自動換行的效果，卡片大小還控制不好

[O] 試做 navbar

  • 除了基本設定與容器內文字置中之外，只要用 media query 設斷點 max-width: 768px
  • 在斷點內對容器設定flex-direction: column; align-items: center;

  • 其他 flex 設定都不用XD
